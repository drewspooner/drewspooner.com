How to Start Playing Dungeons and Dragons

Why should I start?

Dungeons and Dragons is one of the most popular and well known tabletop roleplaying game. There is a low barrier to entry, and it can be enjoyed by anyone! There are a clear structure of rules to guide players, however there is plenty of room for creativity!

How do I get started?

All you really need to start is paper, pencils, dice (which can be physical or found online) and a good attitude. Basic free rule sets and adventures can be found online for anyone to use!

  • Starter Dice Kit (Amazon)
  • Basic Rules (Free)
  • Starter Set (Amazon)

How do I create a character?

There are a limited number of classes, character races, backgrounds, and personality traits, however there are an insanely large number of combinations and an unlimited number of backstories to create for your character. This may seem intimidating, however, a good way to start is to pick your favorite fictional hero and try to create a character that matches them. Once you have experience with that method of character creation, you can more easily start building your own more original characters.
